# CI-CD---GitHub-Actions

Pagina web M3 E10 - CI/CD - GitHub Actions

Desarrollo de una Aplicación Web con CI/CD

Objetivo: Desarrollar una aplicación web utilizando HTML, CSS y JavaScript, y configurarla para que utilice CI/CD con GitHub Actions y despliegue automático en un servidor de nube.

 Requisitos:

Desarrollar una aplicación web: Crear una aplicación web que permita a los usuarios realizar una tarea específica, como un formulario de registro simple.
Configurar CI/CD con GitHub Actions: Configurar GitHub Actions para que se ejecuten pruebas unitarias y de integración automáticamente después de cada commit.
Despliegue automático en un servidor de nube: Configurar el despliegue automático de la aplicación web en un servidor de nube a elegir.
Monitoreo y logging: Configurar monitoreo y logging para la aplicación web para detectar y solucionar problemas.
Entregables:

Código fuente: El código fuente de la aplicación web, incluyendo el HTML, CSS y JavaScript.
Archivo de configuración de GitHub Actions: El archivo de configuración de GitHub Actions que define el flujo de trabajo de CI/CD.
Informe de despliegue: Un informe que describe el proceso de despliegue y cualquier problema que se haya encontrado.
